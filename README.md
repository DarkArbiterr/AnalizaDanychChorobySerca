# Analiza Danych - Choroby Serca

## Dane

Analizie zostały poddane dane pacjentów chorujących lub nie na chorobę serca. Celem analizy jest:

1. Eksploracyjna analiza danych
2. Wybór predyktorow do przewidywania występowania choroby serca
3. Podział danych na zbiór treningowy i testowy
4. Zbudowanie 2 modelów klasyfikacyjnych i ich porównanie (MLP i Lasy Losowe)

## Raport

W pliku **.pdf** znajduje się szczegółowy raport z przeprowadzonej analizy w **języku R**.
